# たねまき アプリのたな（tanemaki-apps）

対話型算数スクール【たねまき】の学習アプリランチャー。保護者と子どもが「今のうちの子に合うアプリ」を探して開くための一覧ページ。

## 対象の子と使いどころ

たねまきの保護者向け。Discordで配信済みのアプリ40本を、領域タブ（かず・けいさん・とけい・はかる）＋年齢目安バッジ＋検索で探せる。困ったときは画面下の「こまったら そうだん」から公式LINEの個別サポへ（アプリは1対1指導の代替、という思想の戻り道）。

## データの編集

`index.html` 内の `APPS` 配列がアプリ一覧の正本。1アプリ1エントリで、ここだけ編集すればよい。

| フィールド | 意味 |
|---|---|
| `name` / `desc` / `url` / `emoji` | カード表示 |
| `cat` | タブ（kazu / keisan / tokei / hakaru） |
| `age` | 年齢目安（「4さい〜」表示） |
| `kw` | 検索用キーワード |
| `chikara` | つくちから（カードにチップ表示・検索対象） |
| `tsumazuki` | つまずきワード（検索対象） |
| `next` | つぎのステップのアプリid |

`chikara` / `tsumazuki` / `next` は「つまずきから探す」機能（次フェーズ）の土台。通常検索はすでにこれらにヒットする。

## 技術

単一 `index.html`・Vanilla JS・外部依存なし。localStorage 不使用・個人情報なし。

## フィードバック収集について

窓口はこのランチャーには置かない（各アプリ内フォーム→スプレッドシート集約の方針。2026-07-03決定）。
